Validating Credential Status

Status checking is defined in the specific revocation method specification at W3C. GS1 Issued credentials use the [[vc-bitstring-status-list]].

GS1 Status Validation Rules

The following rules must be performed for validation credential status:
  • A credentialStatus is valid if the rules in Setion 3.2 of [[vc-bitstring-status-list]] return a valid result of true for each credentialStatus entry.
!

Note:The governance around the meaning of revocation and suspension of credentials are still under development in the GS1 governance system.
