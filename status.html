Status of this document

This section describes the status of this document at the time of its publication.

This document is maintained by an internal group convened by GS1 Global Office that comprises representatives from the GS1 Member Organisations actively involved with implementing Verifiable Credentials.

Although this is not a GS1 standard, it is subject to the IP Policy. Code samples, JSON schemas and JSON-LD context files are subject to the Apache 2.0 license. Comments and contributions to GS1 Digital Licenses are welcome through its GitHub repository, noting the IP policy.

For any questions about this document or Verifiable Credentials at GS1, please contact vcsc@gs1.org
